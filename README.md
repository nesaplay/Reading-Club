# Showcasing 

    -   Javascript (ES6)
    -   React.js
    -   RESTFul API
    -   Routing and communication
    -   Entities

# Stack

    Language:   HTML5, CSS3, JS(ES6)
    Framework: React, MaterializeCSS
    Module Loader: Webpack
    Package Manager: yarn
